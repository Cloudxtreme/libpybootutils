libpybootutils
Bootloader configs parsing libraries for Python. You don't need to waste your time to create GRUB 0.97 config parser because it's already done.

License: This software is distributed on Open and Free Software terms - GPLv3
Requirements: Python 2.x
Source code: libpybootutils
Required by: xsm-py

To download project files you need a Git, then you can execute: 

$ git clone git://github.com/webnull/libpybootutils
To install you just need to run: 
$ ./setup.py build
sudo ./setup.py install

Importing bootloader parser to your application: 

>>> import libpybootutils.grub as grub
>>> print grub
